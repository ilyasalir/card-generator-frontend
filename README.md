# Greeting Card Generator

## Instructions to Run the Project

1. **Clone the repository:**
    ```sh
    git clone https://github.com/your-username/greeting-card-generator.git
    cd greeting-card-generator
    ```

2. **Install dependencies:**
    ```sh
    npm install
    ```

3. **Run the project:**
    ```sh
    npm start
    ```

4. **Test the project:**
    ```sh
    npm test
    ```
